# UGA 演化沙盒

## 这是什么

一个多智能体演化模拟。100-200 个"小人"在一张地图上活着、吃饭、生孩子、死掉。它们的行为由一小坨数字（基因）决定，基因通过生死筛选一代代变好。

我想回答一个问题：**"意图"——有目标地行动——能不能从无到有自己长出来？**

不是人给 agent 编一个目标，而是让自然选择把目标感雕刻出来。

---

## 核心假设（已被否定）

> 资源越稀缺，agent 越需要"动脑子"（规划/记忆）才能活下来。所以高压环境会筛选出有意图的行为。

直觉上很合理。实验跑了五轮，结论是：**错的。**

---

## 五轮实验讲了什么故事

### 第一轮：我把答案写进了出题里

给每个小人一张表："饿了就多干活"。然后观察到它们"涌现"了分工。

问题很明显——这不是涌现，是我写的剧本它照着演了。废弃。

### 第二轮：让聪明人和笨人在同一个镇上竞争

- **聪明人**：每天花 4 块钱买报纸，看看今天镇上什么工作好做，算一下最优选择
- **笨人**：不看报纸，凭祖传直觉行动。但直觉会遗传——爸爸的直觉好活下来了，直觉差的没后代

两种人放在同一个世界里，看谁活得多。

**结果：**
- 穷镇（工作少）：聪明人全死了。每天 4 块报纸钱在穷镇是压死骆驼的稻草
- 富镇（工作多）：聪明人赢了。报纸钱不算什么，精准判断让它们生得更快

**结论：聪明是奢侈品，不是生存必需品。越穷越不需要聪明，因为穷的时候"想"这件事本身就要花你承受不起的钱。**

### 第三轮：那记忆呢？比"聪明"便宜多了

- **有记忆的小人**：脑子里能存上一步发生了什么（递归神经网络）
- **没记忆的小人**：只看眼前，每一刻都是全新的（前馈神经网络）

放在一张会长草的地图上，草吃了会长回来。有季节变化。

**结果：没差别。** 两种人活得一样好。

**原因：** 小人能看到周围 3 格，到处都是草。看一眼就知道下一口饭在哪——根本不需要记住任何东西。记忆是个用不上的器官，白白增加了基因复杂度。

### 第四轮：那就让"看一眼不够"

把环境调狠：
- 冬天完全不长草（不是长得慢，是零）
- 40% 的地图是永久荒漠
- 视野缩到 2 格（只看得见紧邻一圈）

**结果：几乎全灭绝了。** 5 次实验里 4 次种群死光。有记忆的反而死得更快。

**原因：** 环境严酷到所有人都在挣扎，谁先碰运气碰到一组能活的基因谁就赢。有记忆的小人基因比没记忆的长一倍（多了 64 个数字），碰到好组合的概率更低。还没学会用记忆就先饿死了。

---

## 结论

跑了五轮之后我意识到原来的问题问错了。不是"压力会不会逼出聪明"，而是：

**1. 复杂能力只在一个很窄的窗口里能长出来。**

太安逸 → 不需要（看一眼就够了，脑子是浪费）。太严酷 → 养不起（还没学会就死了）。只有"刚好够活、但靠本能又不太够"的中间地带，复杂认知才有选择优势。

**2. 一种能力有没有用，不取决于世界有多难，取决于"没有它时你现有的手段够不够"。**

世界可以很复杂，但如果你看看周围就能做出正确判断，记忆和规划就是多余的。关键不是难度，是信息缺口。

**3. 复杂结构最大的代价不是"用起来贵"，是"学会太慢"。**

递归网络跑一次并不比前馈网络贵。但它的基因有 200 个数字要调对，前馈的只有 130 个。在有限代数里，越简单的结构越容易碰到好组合。

**4. 在稳定环境里，本能几乎总是赢过思考。**

如果世界的规则不变，自然选择能把最优反应直接刻进基因——跳过"思考"这个中间步骤。思考只在"规则经常变、刻进基因来不及"时才值回票价。

**5. 你不可能从你预设的选项里观察到涌现。**

如果系统只能输出你设计好的几种行为，那你看到的不是涌现，是你自己思维的回放。

---

## 方法论

纯演化算法，无外部 API 调用，无神经网络训练框架——所有网络（前馈/递归）都是手写的矩阵乘法，基因就是展平的权重向量。

- 每轮跑 10,000–12,000 tick，种群 100–200 个体，地图 40×50 格
- 选择压力来自死亡（余额归零）和繁殖（余额过阈值才能分裂）
- 变异率固定 0.05，高斯噪声加在权重上
- 对照组在同一次 run 内并行跑（混合种群竞争，或双世界对照）
- v3 用 planner 频率走势 + Welch t-test；v4/v5 用存活率和种群占比

不依赖任何外部库，Python 3.8+ 标准库即可复现。

---

## 已知局限

1. **种群规模小**：100–200 个体，演化搜索空间受限。RNN 的 200 维基因在这个种群规模下可能根本搜不到好解，结论"记忆没优势"有可能只是"搜索预算不够"。
2. **单次 run**：每个条件只跑了一次（v3 跑了两个世界算对照），没有多种子统计。结论的置信区间未知。
3. **地图尺寸固定**：50×50 格 + 视野 2–3 格。如果地图更大、绿洲间距更远，记忆的优势可能会显现——当前设计可能还没把信息缺口拉到够大。
4. **无渐进施压**：v5 直接把三种压力拉满，导致大面积灭绝。缺少"先让种群适应基础生存、再逐步加压"的课程学习设计。
5. **变异算子单一**：只有高斯噪声，没有交叉、基因片段复制等。对高维基因空间的搜索效率可能不够。

---

## 下一步

核心矛盾是：环境要严酷到"必须用记忆"，但不能严酷到"还没学会就死了"。

计划：
- 先让种群在温和环境里演化出基本生存能力（给学费期）
- 然后逐步加压：引入冬天、拉长荒漠
- 设计"必须穿越看不到食物的区域才能到达下一个绿洲"的地形——只有记住方向的小人才能走过去
- 如果有记忆的小人真的演化出了"在看不见食物时仍然朝一个方向持续走"的行为——那就是意图的最小可观察形式
