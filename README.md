# UGA Evolution Sandbox

验证假设 A："资源稀缺 × 规划/认知能力 → 意图涌现"。

通过五轮迭代实验，**假设 A 被否定**，但产出了更有价值的通用结论。

---

## 实验迭代链

### v1 → 循环论证（被废弃）

研究者手写策略权重表（"饿了就多干活"），观察到 serve_specialist 涌现。
问题：答案写在出题里。这不是涌现，是预设的回放。

### v3 → 规划是奢侈品（假设 A 被否定）

设计：50 个 planner（1-step lookahead，每 tick 付代谢税）+ 50 个 evolver（策略向量遗传变异），同一世界竞争。

结果：
- **高压环境：planner 全灭（0%）**
- **低压环境：planner 扩张到 72%**

原因：规划有固定代谢成本，在资源极度稀缺时这笔税致命。演化者靠自然选择直接雕刻出精确本能——不需要实时计算。

### v4 → 看一眼就够时记忆是多余的

设计：统一物理规则（草逻辑斯蒂生长+吃），前馈网络 vs 递归网络混合竞争。

结果：无差异。视野 3 格已经够看到下一口饭，记忆没用武之地。

### v5 → 严酷环境里种群来不及学会就灭绝

设计：冬天 growth=0、40% 荒漠、视野缩到 2 格、绿洲分散。

结果：5 个 seed 里 4 个种群灭绝。RNN 显著劣势——基因搜索空间比 FF 大一倍，在挣扎求生时"碰到好基因"更慢。

---

## 通用结论

1. **复杂性只在一个窄窗口里生长。** 太安逸不需要，太严酷养不起。倒 U 形。

2. **能力有没有用，取决于"没有它时现有能力够不够"。** 不取决于世界本身复不复杂。

3. **复杂结构的真正代价不是"运行贵"，是"学会太慢"。** 搜索空间大 = 在有限世代里更难碰到好组合。

4. **本能在稳定环境中几乎总是赢过实时计算。** 规划只在"环境结构经常变、本能来不及适应"时才不可替代。

5. **你不可能从预设的选项里观察到真正的涌现。** 系统输出空间必须大于研究者的预期空间。

---

## 下一步方向

核心矛盾：环境要严酷到"看一眼不够"才能让记忆有用，但不能严酷到种群活不过学费期。

解法思路：
- 分阶段施压（先让种群在温和环境里演化出基础能力，再逐步加压）
- 缩小 RNN 搜索空间（隐藏层 8→4，减轻学费代价）
- 制造"必须穿越感知盲区"的地形（长走廊连接绿洲）

## 背景

这是 UGA Network 项目的前期验证实验。UGA 的核心假设是"在选择压力下，AI agent 能自发涌现出目标导向行为（意图）"。

本沙盒的结论是：这个假设需要重新措辞——意图涌现的前提不是"压力大"，而是"刚好够活但感知不够用"。
